# Apple Stock Analysis Tool | ACC102 Track4
Student ID: 2469051
Name: Yunfei Zhao

This is an interactive stock analysis tool built with Python and Streamlit. It visualizes Apple (AAPL) stock price trends, daily return distribution, and key financial metrics for the 2024-2026 period.

- Data Provider: AkShare
- Stock Symbol: AAPL (Apple Inc.)
- Time Period: 2024-01-01 to 2026-04-16
- Metrics Included: Daily Open/High/Low/Close Prices, Trading Volume

1. Display the latest 10 days of stock data
2. Visualize the AAPL stock price trend from 2024 to 2026
3. Show the distribution of daily returns
4. Calculate and display key metrics:
   - Latest closing price
   - Average daily return
   - Daily volatility (risk indicator)
